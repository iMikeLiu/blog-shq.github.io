A. Vasya And Password

题目描述

Vasya came up with a password to register for EatForces — a string sss . The password in EatForces should be a string, consisting of lowercase and uppercase Latin letters and digits.

But since EatForces takes care of the security of its users, user passwords must contain at least one digit, at least one uppercase Latin letter and at least one lowercase Latin letter. For example, the passwords abaCABA12, Z7q and 3R24m are valid, and the passwords qwerty, qwerty12345 and Password are not.

A substring of string sss is a string x=slsl+1⋯sl+len−1(1≤l≤∣s∣,0≤len≤∣s∣−l+1)x = s_l s_{l + 1} \cdots s_{l + len - 1} (1 \le l \le |s|, 0 \le len \le |s| - l + 1)x=s​l​​s​l+1​​⋯s​l+len−1​​(1≤l≤∣s∣,0≤len≤∣s∣−l+1) . lenlenlen is the length of the substring. Note that the empty string is also considered a substring of sss , it has the length 000.

Vasya's password, however, may come too weak for the security settings of EatForces. He likes his password, so he wants to replace some its substring with another string of the same length in order to satisfy the above conditions. This operation should be performed exactly once, and the chosen string should have the minimal possible length.

Note that the length of sss should not change after the replacement of the substring, and the string itself should contain only lowercase and uppercase Latin letters and digits.

输入输出格式

输入格式

The first line contains a single integer TTT ( 1≤T≤1001 \le T \le 1001≤T≤100 ) — the number of testcases.

Each of the next TTT lines contains the initial password s ( 3≤∣s∣≤1003 \le |s| \le 1003≤∣s∣≤100 ) , consisting of lowercase and uppercase Latin letters and digits.

Only T=1T = 1T=1 is allowed for hacks.

输出格式

For each testcase print a renewed password, which corresponds to given conditions.

The length of the replaced substring is calculated as following: write down all the changed positions. If there are none, then the length is 000 . Otherwise the length is the difference between the first and the last changed position plus one. For example, the length of the changed substring between the passwords abcdef →\rightarrow→ a7cdEf is 444 , because the changed positions are 222 and 555 , thus (5−2)+1=4(5 - 2) + 1 = 4(5−2)+1=4.

It is guaranteed that such a password always exists.

If there are several suitable passwords — output any of them.

说明

In the second example Vasya's password is ok from the beginning, and nothing has to be changed. That is the same as replacing the empty substring with another empty substring (length 000).
